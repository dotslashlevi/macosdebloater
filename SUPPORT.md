# Support Policy

I will not support people using older versions of MacOS, or older versions of the script, this is for MacOS 10.10-12.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0+    | ✔️                 |
| 0.2     | 🚧                 |
| 0.1     | ❌                 |
## Reporting issues

Report bugs with the Issues tab in the ribbon.
